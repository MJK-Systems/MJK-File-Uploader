# MJK File Uploader

A clean, branded file upload tool built for MJK Systems.  
This Streamlit app provides a simple interface for uploading images through a calm, minimal design.

## Features
- Centered MJK Systems logo
- Clean beige/burgundy/black brand palette
- Drag‑and‑drop upload area
- 200MB file size limit
- Supports JPG, JPEG, PNG
